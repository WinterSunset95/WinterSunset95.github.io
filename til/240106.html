Today I learnt how to dynamically set title and meta headers with Vanilla Javascript.

My TIL page is a single page application, so I needed a way to dynamically set the title and meta headers. Luckily, this was just one google search away.

We can use the document.title property to set the title of the page. And for setting the meta description, we can use the following code: document.querySelector("meta[name='description']").setAttribute("content", "This is my description")
