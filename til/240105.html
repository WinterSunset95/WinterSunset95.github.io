Accessing your PHP development server from other devices on the network

Dated: 05/01/2024

As I was building my website, I wanted to be able to access it from my phone and tablet to see how it looked on them.

I use Archlinux with i3wm as my main OS, and the normal PHP development server to test my code.
Here's what I did:

First, I found my local IP address with the command: ifconfig Output: Look for "inet " under "wlo1" (or whatever your network interface is called). In my case, it's 192.168.243.51

Go to your project directory and run the PHP development server with the command: php -S 192.168.243.51:8080 (Note that you need to use your own IP address here, not mine) Output:

Now, you can access your website from any device on the same network by typing your IP address and port number in the browser.
For example, I can access my website from my phone by typing into the browser: http://192.168.243.51:8080 Output:

And there you have it! You can now access your website from any device on the same network.
