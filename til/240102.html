How to disable pull-to-refresh on mobile devices

Dated: 02/01/2024

I was working on MyOnlinePhone when I came across this problem, pull-to-refresh was conflicting with the swipe gestures.

On most mobile devices, when you swipe down, it refreshes the page. This is called pull-to-refresh. While being a really handy feature for most websites, it can be a problem for websites that use swipe gestures.

You can use overscroll-behavior: none; to disable pull-to-refresh 
	
html, body {
	overscroll-behavior: none;
}

Note that you have to put it on both the html and body elements for it to work on all browsers.

You can also use touch-action: none; to disable all touch gestures. Although I'm not sure what madman would ever build something that requires this.

Never use these as default.
