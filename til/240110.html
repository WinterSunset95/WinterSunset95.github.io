Why Neovim?

Neovim is light, highly extensible and fast. A perfect match for my potato laptop. With emmet-vim, coc.nvim and a bunch of tricks I have learnt over the past couple years, I code at the speed of thought (I am also quite stupid so "speed of thought" isn't all that fast lol).

So today, I learnt how to copy and paste files in Netrw. I will be adding this to my list of tricks.

Working with buffers

  • So you are editing a file, you want to see what other files you have open? :ls
  • You want to cycle through the open buffers? :b {Tab}
  • Jump to a file that is open in another buffer? :b {a part of the filename}{Tab for autocomplete} Suppose I am in index.html and I want to go to my stylesheet which is in styles/main.css, I can type :b main{Tab} Check if it autocompletes, and hit enter. Really useful stuff.
  • Want to close a buffer? :bd {buffername or number} Autocomplete works here too.
  • See what is in your directory and open a new file? :e {Tab} Tab autocomplete is a lifesaver.
  • Switch to the last buffer you were in? Ctrl+Shift+^

Movement using code folding

  • Toggle a fold za
  • Delete a fold zd
    Note: You can see the fold markers on the sidebar
  • Code folding based on indentation? :set foldmethod=indent
  • Manually defined code folding (a personal favourite) :set foldmethod=manual Then you visually select the code you want to fold and type zf. {Visual}zf Personally, I go to the beginning of a tag, Shift + V to Visually select lines, % to jump to the closing tag. And then I make a fold with zf
  • Movement within a fold? [z will move to the start of current open fold and ]z will move to the end of the currently open fold.
  • Movement between folds? zj will move to the start of the next fold and zk will move to the end of the previous fold.

Windows and Tabs

  • Open a file/buffer in a new window? :vsp {filename} :vert sb {buffername}
  • Open a file/buffer in a new tab? :tabnew {filename} :tab sb {buffername}
  • Move current window in a new tab? Ctrl+w Shift+T
  • Cycle between Windows Ctrl+w w
  • Cycle between tabs gt
  • Change window position Ctrl+w {H, J, K, L}
  • Resize windows Ctrl+w {+, -, <, >}

Netrw

  • Open Netrw :Ex
  • New file %
  • New directory d
  • Open file in vertical split v
  • Mark a file(s) mf
  • Copy marked files mc
  • Preview a file p
  • Execute a file/Open in browser x

Miscellaneous

  • Sometimes I have to use vanilla vim. In those situations, I always remap the Escape key to "jj" :inoremap jj <Esc>
  • Quickly open up your terminal Ctrl+z
    $ {run command in your terminal}
    $ fg
    The fg command will bring you back to vim.
  • If you wanna use terminal mode :tabnew +term
  • Visually select the previous text you selected gv
  • Increment the visual selection Ctrl+Shift+a
  • Increment the visual selection (one increment per line) g Ctrl+Shift+a
